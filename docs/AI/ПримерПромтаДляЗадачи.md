## Перед выполненем задачи
Прочитай файл Instructions.md и строго следуй инструкцяим из него.

## 🎯 Основная задача

### Контекст
Напишите тест для конфигурации **ERP**, которая подключена как клиент тестирования.

### Требования к тесту

#### 1️. Ввод начальных остатков товара

| Параметр | Значение |
|----------|----------|
| **Товар** | Материал без характеристики |
| **Склад** | Оптовый неордерный |
| **Документ** | Приобретение товаров и услуг |
| **Поставщик** | Поставщик1 |

#### 2. Проверка отчета

После выполнения пункта 1 необходимо проверить отчет **«Остатки и доступность товаров»**:
- ✅ Должны появиться остатки на складе
- ✅ Соответствие количества введённому в документе

#### 3. Оформление документа "Заказа на перемещение" на половину товара на склад "Магазин неордерный".

#### 4. Выполнить Отгрузку заказа
| Параметр | Значение |
|----------|----------|
| **Документ** | Заказ на перемещение |

Команда Отгрузить в документе или в форме списка документов "Заказ на перемещение"
Затем документ надо Записать.
