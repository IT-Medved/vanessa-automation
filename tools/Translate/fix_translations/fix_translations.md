# Fix Translations — Скрипт для автоматического перевода 1С

## Назначение

Скрипт `fix_translations.py` предназначен для автоматического перевода строк в XML-файлах локализации 1С. Он находит строки со значением `Translation not found` и заменяет их на переведённый текст через Google Translate API.

## Требования

- Python 3.6+
- Доступ к интернету (для работы Google Translate API)
- Установленные модули: `os`, `re`, `urllib`, `json` (входят в стандартную библиотеку Python)

## Установка

Дополнительной установки не требуется. Скрипт использует только стандартные модули Python.

## Использование

### Перевод всех языков

```bash
python fix_translations.py
```

### Перевод конкретного языка

Укажите двухбуквенный код языка в качестве аргумента:

```bash
# Перевод только английского языка
python fix_translations.py en

# Перевод только шведского языка
python fix_translations.py sv

# Перевод только немецкого языка
python fix_translations.py de
```

## Структура файлов

Скрипт обрабатывает файлы в следующей структуре каталогов:

```
locales/Messages/Templates/{lang}/Ext/Template.xml
```

Где `{lang}` — двухбуквенный код языка (например, `en`, `sv`, `de`).

## Как это работает

1. **Поиск файлов** — скрипт сканирует каталог `locales/Messages/Templates/` и находит XML-файлы в подкаталогах `Ext`.

2. **Парсинг XML** — извлекает текст из элементов `rowsItem`, которые содержат:
   - Колонка 0: русский оригинал текста
   - Колонка 1: перевод (где может быть `Translation not found`)

3. **Перевод** — для каждой строки со значением `Translation not found`:
   - Отправляет русский текст в Google Translate API
   - Получает перевод на целевой язык
   - Заменяет `Translation not found` на переведённый текст

4. **Сохранение** — записывает изменения обратно в XML-файл.

## Логирование

Скрипт выводит в консоль:
- Путь к обрабатываемому файлу
- Текст каждой переводимой строки
- Количество выполненных замен и переводов

Пример вывода:
```
Поиск файлов в: c:\Commons\rep\vanessa-automation\locales\Messages\Templates
Целевой язык: en
Режим: онлайн-перевод через Google Translate
Обработка файла: c:\Commons\rep\vanessa-automation\locales\Messages\Templates\en\Ext\Template.xml (язык: en, целевой: en)
  Перевод: - Всего найдено:...
  Перевод: - Выполнение сценариев:...
  Выполнено замен: 150, переведено: 150
```

## Особенности

- **Кэширование переводов** — одинаковые строки переводятся только один раз
- **Задержка между запросами** — 0.5 секунды между запросами к Google Translate API для предотвращения блокировки
- **Экранирование XML** — специальные символы `<`, `>`, `&` автоматически экранируются
- **Обработка Unicode** — проблемные символы заменяются на ASCII-аналоги для корректного отображения в консоли Windows

## Возможные проблемы

### Ошибка доступа к Google Translate API

Если вы получаете ошибку доступа к API, проверьте подключение к интернету и попробуйте увеличить задержку между запросами.

### Некорректное отображение в консоли Windows

Скрипт автоматически заменяет проблемные Unicode-символы. Если вы видите некорректные символы, убедитесь, что консоль поддерживает кодировку UTF-8.

### Файлы не изменяются

Скрипт записывает файл только если были внесены изменения. Если в файле нет строк `Translation not found`, он не будет изменён.

## Внесение изменений

Для изменения параметров скрипта отредактируйте файл `fix_translations.py`:

- `LANGUAGE_MAP` — словарь соответствия кодов языков
- `delay_between_requests` — задержка между запросами к API (по умолчанию 0.5 сек)
- `clean_for_console()` — функция замены Unicode-символов
